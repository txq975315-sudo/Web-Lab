## 一、项目背景与目标

### 产品定位
Thinking Lab作为三层训练系统：
1. **第一层：压力测试**（思维强度训练）- 反复追问，形成项目树
2. **第二层：成长教练**（知识输入+面试模拟）- 案例教学，面试追问
3. **第三层：对话考古**（决策复盘）- 历史对话，人机合作，决策复盘

### 考古V2.0核心设计决策
1. **可迭代分析** - 用户可多次追加对话，AI合并分析，不覆盖已确认内容
2. **混合确认** - AI生成的每条结果有✓确认/✗驳回/✏编辑按钮
3. **智能归档** - AI建议每个条目归档到哪个模板，用户一键确认

### 五维分析架构
```
- 时间轴（timeline）：关键时间节点
- 关键转折点（turningPoints）：决策过程
- 认知盲区（blindSpots）：未考虑的问题
- 假设清单（assumptions）：未验证的假设
- 知识资产（assets）：可归档的洞察
```

#### 组件2：ArchaeologyInputArea（输入区）
**文件：** `src/components/ArchaeologyInputArea.jsx`
- 功能：粘贴对话、追加并分析
- 特点：
  - 调用AI进行五维分析
  - 自动解析JSON结果
  - 自动给条目添加id和status
  - 保留已确认内容，新增待确认内容

```javascript
const handleAddAndAnalyze = async () => {
  // 1. 保存对话片段
  archaeologyStore.addConversationChunk(sessionId, inputText)
  // 2. 获取合并后的对话
  const mergedText = archaeologyStore.getMergedConversation(sessionId)
  // 3. 调用AI分析
  const response = await callAI(ARCHAEOLOGY_V2_PROMPT, mergedText)
  // 4. 解析JSON
  let analysis = JSON.parse(response)
  // 5. 给条目添加id和status
  const addIds = (items) => items.map((item, idx) => ({
    ...item, id: `item_${Date.now()}_${idx}`, status: 'pending'
  }))
  // 6. 更新分析结果
  archaeologyStore.updateAnalysis(sessionId, 'timeline', addIds(analysis.timeline))
  // ...其他维度
}
```

```javascript
const handleArchiveAsset = async (asset) => {
  // 获取或创建项目
  let projects = store.getProjects()
  let projectId = projects.length === 0 
    ? store.createProject('决策复盘项目', '从考古对话中提取的项目').id
    : projects[0].id
  
  // 根据建议模板选择
  let templateKey = 'value_proposition'
  if (asset.suggestedTemplate) {
    const lower = asset.suggestedTemplate.toLowerCase()
    if (lower.includes('persona')) templateKey = 'persona'
    if (lower.includes('value')) templateKey = 'value_proposition'
    if (lower.includes('canvas')) templateKey = 'lean_canvas'
  }
  
  // 创建文档
  store.createDocument(projectId, templateKey, {
    content: asset.content,
    type: asset.type,
    source: '考古对话',
    sourceSession: session.id
  }, `[考古] ${asset.content.slice(0, 30)}...`)
}
```

## 四、完整功能流程

### 从用户视角
```
1. 进入「对话考古」模式
   ↓
2. 点击「+新建考古会话」创建会话
   ↓
3. 在输入区粘贴对话内容
   ↓
4. 点击「追加并分析」
   ↓
5. AI进行五维分析
   ↓
6. 人工对每条结果确认/驳回/编辑
   ↓
7. 点击「报告」按钮查看完整报告
   ↓
8. 对知识资产点击「归档」按钮，归档到项目树
   ↓
9. 可选：导出Markdown报告
```

### 数据流向
```
用户对话输入 
  → localStorage（存储对话片段）
  → AI API（五维分析）
  → localStorage（存储分析结果）
  → UI（显示并允许确认/编辑）
  → 用户确认
  → 报告生成
  → 可选归档到项目树
```
