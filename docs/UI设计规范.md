## 设计理念

### 核心理念
本设计系统以"清新、专业、沉浸式学习体验"为核心设计理念，旨在为用户提供一个视觉上舒适、操作上高效的思维训练环境。通过柔和的色彩、流畅的动效和清晰的信息层次，营造一个有利于专注与深度思考的数字空间。

### 设计原则

| 原则 | 说明 |
|------|------|
| **简洁优先** | 去除冗余元素，保留核心功能，让用户聚焦于学习内容本身 |
| **信息层次清晰** | 通过字号、色彩、间距的差异化处理，建立明确的视觉层级 |
| **视觉一致性** | 全平台统一的设计语言，确保用户在不同场景下获得一致的体验 |
| **舒适阅读** | 合理的字号、行高、对比度，确保长时间使用不易视觉疲劳 |

### 适用范围
本设计规范适用于思维训练工作台全平台 Web 应用，包括但不限于：
- 落地页（Landing Page）
- 选择页（Selection Page）
- 工具页（Tool Page）——包含三栏布局的主工作区
- 各类弹窗、拖曳层、下拉菜单等交互组件

## 字体规范

### 字体家族

| 类型 | 字体 | 备选 | CSS变量 |
|------|------|------|---------|
| 中文 | Microsoft YaHei（微软雅黑） | PingFang SC | `--font-chinese` |
| 英文/数字 | Arial | Helvetica Neue | `--font-english` |
| 等宽/代码 | Consolas | Monaco | `--font-mono` |

### 字号层级

| 层级 | 字号(px) | 字重 | 行高 | 用途 |
|------|----------|------|------|------|
| H1 大标题 | 28-32px | Bold (700) | 1.4 | 落地页主标题 |
| H2 模块标题 | 20-24px | SemiBold (600) | 1.5 | 页面区块标题 |
| H3 卡片标题 | 16-18px | Medium (500) | 1.5 | 卡片内标题 |
| Body 正文 | 14px | Regular (400) | 1.6 | 主要内容文字 |
| Caption 辅助 | 12px | Regular (400) | 1.5 | 标签、注释、时间戳 |
| Tiny 极小 | 10-11px | Regular (400) | 1.4 | 徽标、角标 |

### 字重规范

| 字重 | 用途 |
|------|------|
| Regular 400 | 正文内容，适用于大段落文字和描述性内容 |
| Medium 500 | 卡片标题、导航项，用于需要轻微强调的场景 |
| SemiBold 600 | 模块标题、按钮文字，用于需要明确强调的交互元素 |
| Bold 700 | 页面大标题、品牌名，用于最高层级的视觉引导 |

## 间距系统

### 基础间距单位

采用 4px 基础网格系统，所有间距均为 4px 的整数倍。

| 名称 | 数值 | CSS变量 | 说明 |
|------|------|---------|------|
| xs | 4px | `--space-xs` | 极小间距，用于微调元素间距 |
| sm | 8px | `--space-sm` | 小间距，用于紧密关联元素 |
| md | 12px | `--space-md` | 中等间距，常用于组件内部 |
| lg | 16px | `--space-lg` | 大间距，标准组件间距 |
| xl | 24px | `--space-xl` | 较大间距，区块分隔 |
| 2xl | 32px | `--space-2xl` | 大区域分隔 |
| 3xl | 48px | `--space-3xl` | 页面级别分隔 |

### 组件内间距

| 组件 | 内边距 | CSS变量 |
|------|--------|---------|
| 按钮 | 8px 16px | `--btn-padding` |
| 卡片 | 20px-24px | `--card-padding` |
| 输入框 | 10px 14px | `--input-padding` |
| 导航栏 | 0 24px | `--navbar-padding` |

### 组件外间距

| 场景 | 间距 | 说明 |
|------|------|------|
| 卡片之间 | 16-20px | 网格间距 |
| 区块之间 | 24-32px | 模块分隔 |
| 页面边距 | 24-32px | 内容区左右边距 |

## 动效规范

| 类型 | 时长 | 缓动函数 |
|------|------|---------|
| 面板展开/收起 | 200-300ms | ease-in-out |
| 页面切换 | 300ms | ease-out |
| 弹窗出现 | 200ms | ease-out |
| 快速交互 | 150ms | ease |

### 推荐动效

| 动效 | 参数 |
|------|------|
| 侧边栏展开 | 从左滑入，200ms ease-out |

| 按钮点击 | scale(0.98)，100ms ease |
