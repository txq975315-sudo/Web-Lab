# ThinkingLab 设计系统使用指南

## 快速开始

### 1. 引入样式文件

在项目入口文件中引入设计系统CSS：

```jsx
// src/main.jsx 或 src/index.jsx
import './styles/thinking-lab-design-system.css';
import './styles/anthropic/base.css'; // 基础样式（已有）
import './styles/workbench-prd.css';   // 工作台样式（已有）
import './index.css';
```

### 2. 使用CSS变量

所有颜色、间距、字体都使用CSS变量定义，不要硬编码色值：

```css
/* ✅ 推荐 */
.btn {
  background: var(--color-accent-orange);
  color: var(--color-text-inverted);
  padding: var(--space-3) var(--space-6);
  border-radius: var(--radius-lg);
}

/* ❌ 不推荐 */
.btn {
  background: #D97757;
  color: #FAF9F5;
  padding: 12px 24px;
  border-radius: 16px;
}
```

## 页面实现指南

### 落地页（Hero）

#### 基本结构
```jsx
import { Cloud } from './DesignSystemExample';

function LandingPage() {
  return (
    <div className="sky-cloud-bg min-h-screen flex flex-col">
      {/* 云朵背景层 */}
      <div className="cloud-layer">
        <Cloud size="lg" style={{ top: '10%', left: '5%' }} />
        <Cloud size="md" style={{ top: '15%', right: '10%' }} />
        <Cloud size="lg" style={{ bottom: '20%', left: '15%' }} />
      </div>
      
      {/* 顶栏 */}
      <header className="tl-top-header relative z-10">
        <div className="tl-header-left">
          <div className="tl-logo">ThinkingLab</div>
        </div>
        <div className="tl-header-right">
          <button className="tl-btn-ghost">登录</button>
        </div>
      </header>
      
      {/* Hero内容 */}
      <main className="flex-1 flex items-center justify-center relative z-10">
        <div className="text-center tl-fade-in-up">
          <h1>思维训练工作台</h1>
          <p>你的刻意练习圣地</p>
          <button className="tl-btn-primary">立即开始</button>
        </div>
      </main>
    </div>
  );
}
```

#### 注意要点
- 云朵层使用 `position: absolute` 和 `pointer-events: none`
- 内容区域需要 `relative z-10` 保证在云朵之上
- 使用 `tl-fade-in-up` 类添加入场动画

### 功能选择页（Dashboard）

#### 基本结构
```jsx
import { Cloud } from './DesignSystemExample';

function Dashboard() {
  const features = [
    { icon: '💪', title: '压力训练', desc: '...' },
    { icon: '🧠', title: '成长教练', desc: '...' },
    { icon: '🔍', title: '对话考古', desc: '...' }

  return (
    <div className="sky-cloud-bg min-h-screen">
      {/* 云朵层 */}
      <div className="cloud-layer">
        <Cloud size="lg" style={{ top: '8%', left: '3%' }} />
        <Cloud size="md" style={{ top: '12%', right: '8%' }} />
      </div>
      
      {/* 顶栏 */}
      <header className="tl-top-header relative z-10">
        {/* ... */}
      </header>
      
      {/* 功能卡片网格 */}
      <main className="flex-1 flex items-center justify-center relative z-10 px-6">
        <div className="tl-feature-grid w-full">
          {features.map((feature, i) => (
            <button 
              key={feature.title}
              className="tl-feature-card"
              style={{ animationDelay: `${i * 80}ms` }}
            >
              <div className="tl-feature-icon">{feature.icon}</div>
              <h3 className="tl-feature-title">{feature.title}</h3>
              <p className="tl-feature-desc">{feature.desc}</p>
              <button className="tl-btn-primary">立即进入</button>
            </button>
          ))}
        </div>
      </main>
    </div>
  );
}
```

## 动画使用

### 淡入上移动画

```jsx
// 单个元素
<div className="tl-fade-in-up">
  内容
</div>

// 依次入场
<div className="tl-fade-in-up tl-delay-1">第一个</div>
<div className="tl-fade-in-up tl-delay-2">第二个</div>
<div className="tl-fade-in-up tl-delay-3">第三个</div>
<div className="tl-fade-in-up tl-delay-4">第四个</div>
```

### 自定义动画

```css
@keyframes customFade {
  from {
    opacity: 0;
    transform: scale(0.95);
  }
  to {
    opacity: 1;
    transform: scale(1);
  }
}

.my-animation {
  animation: customFade var(--duration-normal) var(--ease-default);
}
```

## 常见问题

### Q: 如何自定义云朵背景？

A: 可以通过 `Cloud` 组件调整大小和位置，或使用CSS实现：

```jsx
// 方式1：使用Cloud组件
<Cloud size="lg" style={{ top: '10%', left: '5%', opacity: 0.8 }} />

// 方式2：纯CSS
<div 
  className="cloud"
  style={{
    position: 'absolute',
    top: '10%',
    left: '5%',
    width: 200,
    height: 100,
    background: 'rgba(255,255,255,0.85)',
    borderRadius: '50%',
    filter: 'blur(2px)'
  }}
/>
```

### Q: 如何调整毛玻璃效果？

A: 修改CSS变量或直接覆盖样式：

```css
/* 调整模糊度 */
.my-glass {
  backdrop-filter: blur(20px);
  -webkit-backdrop-filter: blur(20px);
}

/* 调整透明度 */
.my-glass {
  background: rgba(253, 252, 248, 0.7);
}
```

### Q: 如何在现有项目中逐步迁移？

A: 建议按以下步骤：

1. 先引入样式文件，不修改现有代码
2. 新页面使用新设计系统
3. 逐步重构旧组件
4. 使用组件示例作为参考
